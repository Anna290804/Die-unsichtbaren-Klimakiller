// Die Täter

Was deine
Website ausbremst

Jede dieser Schwachstellen erhöht den Energieverbrauch — und damit den CO₂-Ausstoß — deiner Website messbar.

🖼️
01
Unkomprimierte Bilder

JPEGs ohne Optimierung, PNG statt WebP, fehlende Lazy Loading — Bilder machen 60–80% der Datenmenge aus und sind der größte Energiefresser.

🧱
02
Schlechte Website-Struktur

Unnötige DOM-Tiefe, redundante HTML-Elemente, fehlende Semantik — schlechter Code zwingt den Browser zu mehr Rechenarbeit und mehr Serveranfragen.

🔍
03
Schlechtes SEO

Fehlende Meta-Tags zwingen Crawler zu wiederholten Besuchen. Doppelte Inhalte, broken Links und fehlende Sitemaps verursachen unnötige Server-Anfragen.

⏳
04
Lange Ladezeiten

Jede Sekunde längere Ladezeit = mehr Serveranfragen, mehr Datentransfer, mehr Energie. Nutzer mit langsamen Verbindungen verbrauchen bis zu 10× mehr Energie.

📦
05
Aufgeblähtes JavaScript

Ungeminified JS, ungenutzte Libraries, kein Tree-Shaking — je mehr Code geladen und ausgeführt wird, desto mehr CPU, desto mehr Akku, desto mehr CO₂.

🗄️
06
Kein Caching / CDN

Ohne Caching wird jede Seite neu geladen. Ohne CDN reist jedes Byte vom Server zum Nutzer über halbe Kontinente — ineffizient und energieintensiv.
